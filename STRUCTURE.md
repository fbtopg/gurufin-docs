# Gurufin GitBook Structure Documentation

## Overview

This GitBook documentation provides a comprehensive guide to the Gurufin ecosystem, covering high-level concepts, technical specifications, and developer resources.

## Documentation Structure (28 pages)

### 1. Introduction (3 pages)
- **Vision & Mission**: The foundational purpose and goals of Gurufin
- **What is Gurufin?**: Overview of the ecosystem and its two-pillar architecture
- **Roadmap**: Development phases from foundation to global scale

### 2. Gurufin Chain - Layer 1 (7 pages)
- **Protocol Overview**: Purpose-built Layer-1 for FX and DeFi
- **Network Architecture**: Tendermint BFT + DPoS consensus engine
- **Interoperability**: IBC, EVM Gateway, and GatewayGX Module
- **Guru-PEG**: Fiat-indexed, predictable transaction fees
- **Tokenomics ($GXN)**: Native token utility and supply allocation
- **Governance**: On-chain decision-making framework
- **Validator Guide**: Requirements, rewards, and operational standards

### 3. GX Chain - Stablecoin Framework (5 pages)
- **GX Overview**: Sovereign stablecoin network architecture
- **Reserve & Backing**: 1:1 fiat backing and proof-of-reserves
- **Mint & Burn Mechanism**: Automated bank-API-anchored issuance
- **Multi-Currency Support**: Jurisdictional sovereignty and cross-chain FX
- **Compliance & Regulation**: PoA consensus and embedded compliance

### 4. GuruDex - OPRS Exchange (5 pages)
- **DEX Overview**: Oracle Priced Reserve Swap architecture
- **Hybrid Pools**: Combined institutional and retail liquidity
- **Liquidity & Rewards**: LP mechanics and fee distribution
- **Smart Contract Logic**: Core contracts and swap routing
- **Risk Mitigation**: Four-layer defense strategy

### 5. Developer Resources (4 pages)
- **Testnet Access**: Network configuration and endpoints
- **API Reference**: Cosmos SDK and EVM JSON-RPC endpoints
- **Full Developer Docs**: Link to comprehensive documentation portal
- **Ecosystem Grant Program**: Funding for builders

### 6. Use Cases (4 pages)
- **Cross-Border Payments**: Remittance and B2B payment solutions
- **Stablecoin FX Trading**: Trading, arbitrage, and derivatives
- **Institutional DeFi**: Tokenized assets and collateral management
- **Government & CBDC**: Central bank and FMI applications

5. **Practical Focus**: Includes concrete examples and actionable guidance
