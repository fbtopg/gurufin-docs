# Tokenomics ($GXN)

GXN is the native utility token powering the Gurufin Chain and GX Stablecoin networks. The maximum supply at genesis is 100,000,000,000 GXN.

**Core Utility**
* **Staking:** Securing the DPoS network via validator delegation.
* **Governance:** Voting on protocol upgrades and treasury allocations.
* **Fee Payment:** Execution of network transactions via Guru-PEG.
* **Liquidity:** Provisioning for GuruDex OPRS trading infrastructure.

**Supply & Allocation**
The 100 Billion GXN supply is distributed across key initiatives: Node Pool (25%) for validator rewards, Ecosystem Development fund (20%) for builders and grants, with the remainder allocated to the team, strategic partners, and community programs under multi-year vesting schedules.

**Economic Sustainability**
A percentage of all collected GXN transaction fees is permanently burned by routing it to the Gurufin Abyss Ledger (GAL), a keyless on-chain address. Validators and delegators are compensated via the Node Pool and an increasing share of network transaction fees.
