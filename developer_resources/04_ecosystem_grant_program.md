# Ecosystem Grant Program

Gurufin Foundation supports builders creating applications, tools, and infrastructure on Gurufin Chain. The grant program provides funding for projects that expand the ecosystem.

---

**Eligible Projects**

| Category | Examples |
|----------|----------|
| DeFi | DEX integrations, lending protocols, yield aggregators |
| Infrastructure | Indexers, oracles, bridges, developer tooling |
| Applications | Wallets, payment solutions, analytics dashboards |
| Education | Tutorials, documentation, community resources |

---

**Application Process**

1. **Submit Proposal** — Describe your project, team, milestones, and funding request
2. **Review** — Foundation team evaluates feasibility and ecosystem fit
3. **Approval** — Accepted projects receive milestone-based funding
4. **Delivery** — Complete milestones and receive subsequent tranches

---

**Contact**

For grant inquiries, reach out via the official channels listed on [gurufin.io](https://gurufin.io).
